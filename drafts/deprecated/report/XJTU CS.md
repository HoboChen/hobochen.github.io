# 概述

各位读者，我们必须首先给出一个观点——西安交通大学对于计算机相关专业的本科教育是不那么完美的；不论是从过程看，包括教学质量、考核质量、作业质量，还是从结果看——学生毕业之后的出路好坏。
虽然说大环境并不完美，但由于大学生活的特殊性——你对你自己的生活有了前所未有的掌控力，你仍然可以凭借聪明或/和努力来获得相对较好的未来。

一个简单的公式：

工作量 = 工作效率 * 工作时间

各位读者，在你毕业之前，你所拥有的选择大体上由若干个方面的累计工作量决定。
工作效率往往决定了你的上限，因为每个人的时间都是有限的，一个人一天通常只能有效率的工作10-14小时。而工作时间往往决定了你的下限，因为大多数人的工作效率都是差不多的（尤其是你们还经过了高考这个统一的筛选）。
