# 动态规划

## 什么是动态规划

动态规划是一种解决问题的思想与方法。

使用动态规划解决问题的前置条件是**重复子问题**和**无后效性**。
它将问题随某种外在条件变化而变化的实在归纳为**状态**，并且总结出随之变化的规律即**转移**。

动态规划通常有较多的思维量，较少的编码量。它是算法中较难的一部分。
