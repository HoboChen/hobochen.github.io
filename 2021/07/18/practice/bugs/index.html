Interesting Bugs

Posted on 2021-07-18 |

These are the interesting (read: annoying) bugs I met.

MSI BIOS Flashing without CPU Requires the Flash Disk To Be MBR Partitioned

But there is no documentation indicates that. And, there are few people met that, as barely no results using Google.

Solution: Make it MBR.

Wasted time: 2h.

Radeon 6800XT Won’t Wake Up after S3 Sleep

Many people met this.

Workaround: diable S3 sleep.

Wasted time: 5h.

Asrock X470D4U Appears POST Failure

I get a black screen after power on. I waitted for several minutes, still black.

Cause: It is a server motherboard, it takes a long time for memory check and etc. And it will wait BMC for around 100 secs. You need to be patient enough.

Wasted time: 5h.

Dell XPS 7590 No S3 Sleep and High Battery Drain for Modern Standby

Wasted time: 20h+.
