冷漠

Posted on 2022-11-26 Edited on 2022-12-18

有段时间我的微信朋友圈签名是：‘我力求客观，但绝不冷漠’。

它是Wright Mills说的（他是《社会学的想象力》的作者），也曾很长一段时间里面是我对自己的要求 -- 直到我发现它实在太难了。总有些事情在我的能力之外，比如，没有好用的中文搜索引擎，比如总会有人违反交通规则，比如总会有人受到小概率事件的影响，它们中的绝大多数都可以让我保持舒适的冷漠。

但有一些事情不能，当我听说重病的病人被医院挡在门外的时候，当我听说我的同事们被封控在家、近乎断炊的时候，当我听说深夜大巴翻车的时候，当我听说消防车开不进小区的时候，我都感觉奇怪、费解、遗憾、痛心。我能做什么呢？我们能做什么呢？

可能的话捐一点点钱或者出一点力（虽然这不太可靠）去帮助受影响的人，在自己的生活被影响的时候吵架，在朋友圈/微博转发评论（虽然说很快就被封了）和阴阳怪气，和朋友们讲这些事情本不应该这样，记住那些曾经大声疾呼、奔走相告的人们。最重要的是，经常让自己的良心告诉自己：哪些是对的、哪些不是、哪些是概率导致的不幸、哪些是愚蠢的人导致的不幸。

Why Do I Prefer Linux

Posted on 2020-01-25 Edited on 2022-10-16

Days ago, I installed Ubuntu 18.04.3 LTS in my thinkpad and intel nuc. Surprisingly, in 2020, I found that quite attractive comparing to other OS when coding and reading papers.

Pros:

  • quiet, responsive, much less disturbing
    • completely no windows defender, mac app store update, or anything other than the OS kernel running in backgroud but toublesome to shutdown
    • theoretically no rebooting required by the OS as no windows update or anything like that
    • fan spins much less comparing to windows
  • good command line;
    • though i can run ubuntu in the windows hyper-v or use wsl; there do exist some need that interacting with the local filesystem or some system component like that
    • i am not familiar with powershell
  • reliable
    • simple privilege management methods provided by the OS
    • much less unintentional waking up from sleep

Cons:

  • no onenote
  • no onedrive
  • more bugs for daily uses comparing to windows
